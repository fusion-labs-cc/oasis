OASIS 瀏覽器擴充功能 — 安裝說明
==================================

這是一個 Chrome / Edge 擴充功能，用來儲存目前分頁的網址與時間，
並可在一般視窗與無痕視窗之間共用清單。


一、安裝步驟
------------

1. 把這個 zip 解壓縮成一個資料夾（裡面會有 manifest.json、popup.html 等檔案）。
   ※ 請保留整個資料夾，不要只留單一檔案。

2. 開啟瀏覽器，在網址列輸入並前往：
       chrome://extensions
   （Edge 使用者請輸入 edge://extensions）

3. 打開右上角的「開發人員模式 / Developer mode」開關。

4. 點「載入未封裝項目 / Load unpacked」，選擇步驟 1 解壓出來的那個資料夾。

5. 工具列會出現 OASIS 的貓咪圖示，安裝完成。
   （若沒看到，點工具列的拼圖圖示，把 OASIS 釘選起來。）


二、開啟無痕模式支援（重要）
----------------------------

要讓「跨無痕視窗共用清單」以及「用無痕視窗開啟已存網址」正常運作，
必須額外授權：

1. 回到 chrome://extensions，找到 OASIS，點「詳細資料 / Details」。
2. 開啟「允許在無痕模式下執行 / Allow in Incognito」。

沒有開這個開關，瀏覽器就不會讓擴充功能在無痕視窗執行，
也無法用無痕視窗開啟清單裡的網址。


三、使用方式
------------

- 點工具列的 OASIS 圖示會跳出清單，列出所有已儲存的網址與儲存時間。
- 按「Save current tab」把目前分頁存進清單。
- 點清單裡的網址 → 一律用無痕視窗開啟。
- 眼睛圖示可隱藏 / 顯示整份清單（隱藏時仍可儲存目前分頁）。
- 清單資料存在瀏覽器本機，不會上傳任何伺服器。


四、更新
--------

下載新版 zip，解壓後覆蓋原資料夾，再到 chrome://extensions
點該擴充功能的「重新載入 / Reload」即可。已儲存的清單會保留。
